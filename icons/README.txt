REQUIRED ICON FILES — create these before publishing

icon-192.png         192x192 px   App icon (home screen, small)
icon-512.png         512x512 px   App icon (Play Store listing)
icon-512-maskable.png  512x512 px   Same image but with safe zone padding (Android adaptive icon)
screenshot-portrait.png  480x800 px   Game screenshot for PWABuilder / Play Store

HOW TO CREATE ICONS (free options):
1. Canva (canva.com) — make a 512x512 design, export PNG, resize copies
2. GIMP — draw/generate, export at each size
3. realfavicongenerator.net — paste a 512x512 PNG, it generates all sizes

MASKABLE ICON SAFE ZONE:
  The center 80% of the image is visible on all Android shapes.
  Keep the important art within a centered circle of ~410x410 px on a 512x512 canvas.
  Background should fill the ENTIRE 512x512 (no transparency).

SUGGESTED ICON STYLE:
  Background: #050510 (dark navy)
  Text/graphic: neon spiral in #7b2fff (purple) or #00ffff (cyan)
  Font: Orbitron Bold (matches the game)
